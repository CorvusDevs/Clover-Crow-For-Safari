Privacy Policy

Clover Crow for Safari. Last updated 2026-05-13

TL;DR. Clover Crow does not collect, transmit, or share any personal data. Every setting and every cached value stays on your device. There is no analytics, no telemetry, no account, and no server.

What is Clover Crow?

Clover Crow is a Safari Web Extension that enhances the 4chan browsing experience with 40+ modules: thread watcher, quote previews, inline image expansion, a quick-reply box, regex filters, extended embeds, and more. It runs entirely inside your browser; the host app exists only to install the extension into Safari.

What data Clover Crow collects

None that ever leaves your device. Clover Crow stores the following on your device using Safari's local extension storage:

  • Module preferences: which features you have enabled and any options you have customized (e.g. theme, quick-reply transparency, autoupdate interval).
  • Per-thread state: counts of posts you have seen so the unread badge works (kept up to 30 days).
  • Watched threads and saved drafts: only if you actively use the Thread Watcher or Saved Drafts modules.
  • UI state: your selected display language, whether the review prompt has been dismissed, popup-open count for triggering that prompt.

All of this lives in Safari's browser.storage.local sandbox. It is never read by Clover Crow's developer, and it is never sent to any server.

Network access

Clover Crow's content scripts only run on 4chan.org / 4channel.org pages. The extension makes additional network requests only to 4chan's own public endpoints (a.4cdn.org, i.4cdn.org, s.4cdn.org) for thread updates and image expansion, the same endpoints your browser already hits when you load 4chan normally.

Optional features that can route traffic to third-party services, all off by default:

  • Translate Post: opens a translation URL (Google / Bing / Yandex / Papago) in a new tab when you click the per-post translate button. Their privacy practices are governed by their own policies.
  • Resurrect Dead Quotes: links dead quotes to third-party 4chan archive sites (e.g. archived.moe).
  • Extended Embeds: when you click an embed button, the embed iframe loads from the original host (YouTube, Vimeo, Imgur, etc.).

Permissions

Safari grants Clover Crow the following permissions:

  • storage: to save your preferences locally.
  • Host access to 4chan.org / 4channel.org, so the extension can modify 4chan pages.
  • Host access to a.4cdn.org, i.4cdn.org, s.4cdn.org, 4chan's official JSON, image, and static-asset endpoints.

Third-party services

Clover Crow does not embed analytics SDKs, crash reporters, advertising libraries, or social-media trackers. No third-party JavaScript is loaded at runtime unless you explicitly invoke an opt-in feature listed above.

Children's privacy

Clover Crow does not knowingly process data from children. Since 4chan's own terms require users to be at least 18, and Clover Crow only operates on 4chan, this requirement is enforced upstream.

Changes to this policy

If this policy ever changes, the "Last updated" date at the top will reflect the new revision. Material changes will also be mentioned in the extension's release notes.
