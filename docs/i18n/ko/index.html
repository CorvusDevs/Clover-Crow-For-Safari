Clover Crow

4chan 을 40 개 이상의 모듈로 향상시키는 Safari 확장 프로그램입니다.

앱 스토어에서 다운로드

어떤 기능을 하는지

Clover Crow 는 Safari 에서 4chan-X 기능을 제공합니다. 모든 보드를 열면 Clover Crow 가 호버 미리보기, 인라인 이미지 확장, 읽지 않은 게시글 수를 표시하는 스레드 감시자, 드래그 가능한 빠른 응답 상자, 정규식 필터, 번역 버튼, 확장된 임베드 등 다양한 기능을 제공합니다.

빠른 응답끌 수 있고 크기를 조절할 수 있는 답글 상자에 초안 자동 저장과 한 번의 탭으로 인용 기능을 제공합니다.
인라인 이미지 확대미리보기 이미지를 클릭하면 해당 이미지나 비디오가 자리에서 완전히 확대됩니다.
스레드 감시자즐겨찾기 스레드를 표시하고, favicon 및 팝업에 읽지 않은 수를 배지로 표시합니다.
인용 스레딩레딧 스타일의 들여쓰기 답변; 누가 누구를 인용했는지 보여주는 중첩된 보기
빠른 필터이름, 트립코드, ID, 이미지 MD5, 또는 파일명으로 한 번에 음소거합니다.
게시글 번역구글, 빙, 야ндекс, 파파고를 통해 게시글별 번역 버튼을 제공합니다.
확장된 임베드YouTube, Vimeo, Streamable, Twitch, Imgur, Reddit, Bitchute 에서 지원됩니다.
33 개 언어전체 로컬라이즈된 UI 와 자동어명 드롭다운 피커를 제공합니다.

개인정보

클로버 크라운은 브라우저 내에서 완전히 실행되며, 브라우저 데이터를 수집하거나 전송하거나 공유하지 않으며 모든 설정은 사용자의 기기에서 저장됩니다. 전체 개인정보처리방침 읽기.

정보

클로버 크로우(Clover Crow) 는 독립 프로젝트이며, 4chan 나 그 운영자와의 관계가 없으며 지원받거나 후원받지 않았습니다. 여기서는 이 확장 프로그램이 향상시키는 플랫폼을 식별하기 위해 "4chan"이라는 이름을 설명적으로 사용하고 있습니다.

Clover Crow 는 에서 개발되었습니다. Corvus 개발자.
