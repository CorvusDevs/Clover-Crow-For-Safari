Clover Crow
for Safari

4chan を 40 以上のモジュールで強化する Safari 拡張機能です。

App Store で入手
macOS 14.0 以降、iOS/iPadOS 17.0 以降に対応しています。
機能

Safari 向けに再構築された必要なすべて

素早い返信

ドラッグして移動可能でサイズ変更可能な返信ボックスで、ドラフトの自動保存とワンタップで引用が可能です。

インライン画像の拡大

サムネイルをクリックすると、その場で画像や動画を拡大表示します。

スレッドモニター

ブックマークスレッドを登録し、ファビコンとポップアップで未読カウントを表示します。

引用スレッド

Reddit 風のインデント付き返信で、誰が誰を引用しているかを階層表示します。

クイックフィルター

名前、トリップコード、ID、画像の MD5、またはファイル名でワンクリックでミュートします。

投稿を翻訳

ポストごとの翻訳ボタン（Google、Bing、Yandex、Papago）

拡張埋め込み

YouTube、Vimeo、Streamable、Twitch、Imgur、Reddit、Bitchute から動画をインポートできます。

33 言語

完全なローカライズされた UI で、自名ドロップダウンピッカーが利用可能です。

モダン化されたカタログ

丸みを帯びたサムネイル、ホバー時のステート、テーマに合わせたコントロールを備えた、清潔感のあるカード形式のカタログです。

引用プレビューとバックリンク

引用をホバーしてリンクされた投稿をプレビューし、バックリンクリストで各投稿の返信を表示します。

ID 投稿数

投稿者 ID の横に投稿数バッジを追加し、クリックすると同じ著者の全投稿をハイライトします。

URL をリンク化

投稿内のリンクをアンカーとしてクリック可能にする機能を、4chan でデフォルトで無効になっているため設定に追加しています。

カスタムテーマ

4chan のボード、カタログ、返信フォームを再スタイリングする、一貫したダークまたはライトのスキンです。

ユニバーサルアプリ

1 回の購入で Mac、iPad、iPhone がすべて利用でき、別プラン、サブスクリプション、アプリ内購入はありません。

瞬時オン/オフ

すべてのモジュールにトグルを備えたマスタースイッチで、アンインストールせずに使用しない機能をオフにできます。

アップデートの新機能

アップデート後の最初の起動時に表示されるようすで、変更点を示すようすで、新規に追加された設定には「NEW」というラベルが表示されます。

仕組み

4chan をより良くする 3 つのステップ

1

App Store からインストール

Clover Crow をダウンロード。ファミリー共有で 1 回の購入で Mac、iPhone、iPad を使用できます。

2

Safari で有効にする

Safari の設定で「機能拡張」を開き、「Clover Crow」を有効にして 4chan へのアクセスを許可します。

3

ボードを開く

4chan を通常通り閲覧し、Clover Crow でプレビュー、ウォッチャー、素早い返信などの機能を追加します。設定ですべてを調整できます。

なぜ Clover Crow を選ぶのか

Safari 固有の完全なツールキット

パワーユーザー優先

Safari 向けに最適化されたクローバー・クロウは、引用プレビュー、ウォッチャー、素早い返信、フィルターなど、長年ブラウザを愛用するパワーユーザーが期待する機能セットを備えています。

データ収集なし

アカウント、テレメトリ、クラウドは不要です。フォロースレッド、フィルター、設定はすべてデバイス上に保存されます。

完全にローカライズ済み

アプリ内で切り替えることができる 32 の言語に対応した機能の説明です。

macOS 14.0 (Sonoma) 以降、または iOS/iPadOS 17.0 以降、Safari 17 以降で動作します。
