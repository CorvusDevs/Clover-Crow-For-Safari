Clover Crow

4chan を 40 以上のモジュールで強化する Safari 拡張機能です。

App Store で入手

機能説明

Clover Crow は Safari に 4chan-X の体験をもたらします。任意のボードを開くと Clover Crow が機能を引き継ぎ、引用のホバープレビュー、インライン画像の展開、未読カウントを表示するスレッド監視、ドラッグ可能な素早い返信ボックス、正規表現フィルター、翻訳ボタン、拡張埋め込みなど、多数の機能を備えています。

素早い返信ドラッグして移動可能でサイズ変更可能な返信ボックスで、ドラフトの自動保存とワンタップで引用が可能です。
インライン画像の拡大サムネイルをクリックすると、その場で画像や動画を拡大表示します。
スレッドモニターブックマークスレッドを登録し、ファビコンとポップアップで未読カウントを表示します。
引用スレッドReddit 風のインデント付き返信で、誰が誰を引用しているかを階層表示します。
クイックフィルター名前、トリップコード、ID、画像の MD5、またはファイル名でワンクリックでミュートします。
投稿を翻訳ポストごとの翻訳ボタン（Google、Bing、Yandex、Papago）
拡張埋め込みYouTube、Vimeo、Streamable、Twitch、Imgur、Reddit、Bitchute から動画をインポートできます。
33 言語完全なローカライズされた UI で、自名ドロップダウンピッカーが利用可能です。

プライバシー

Clover Crow はブラウザ上で完全に動作し、閲覧データを収集、送信、共有することはありません。すべての設定はお使いのデバイスに保存されます。 完全なプライバシーポリシーを参照してください.
