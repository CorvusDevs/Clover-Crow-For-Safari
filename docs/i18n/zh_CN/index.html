Clover Crow
for Safari

一款为 4chan 增强功能的 Safari 扩展，包含 40 多个模块。

重要：这些英文单词未翻译，请完整翻译为简体中文：应用，商店。获取应用商店
适用于 macOS 14.0 及以上版本和 iOS/iPadOS 17.0 及以上版本。
功能

为 Safari 重新打造的所需一切

快速回复

支持拖拽和缩放的回复框，具备草稿自动保存和一键引用功能。

行内图片扩展

点击缩略图即可原地展开完整图像和视频

线程监视器

收藏帖子，在浏览器图标和弹出窗口上显示未读计数徽章。

引用线程

Reddit 风格的缩进回复；显示引用关系的嵌套视图

快速筛选

通过名称、旅行代码、ID、图像 MD5 或文件名一键静音。

翻译帖子

通过 Google、Bing、Yandex 或 Papago 为每篇文章提供翻译按钮。

扩展嵌入

支持 YouTube、Vimeo、Streamable、Twitch、Imgur、Reddit 和 Bitchute。

33 种语言

支持自动选择本地化名称的全本地化界面

现代化目录

这是一个采用圆角缩略图、悬停效果及主题化控件的整洁卡片式目录。

引用预览和反向链接

将鼠标悬停在任何引用上即可在行内预览关联帖子，回链列表显示每个帖子下方的所有回复。

帖子计数

在作者 ID 旁添加帖子计数徽章，点击该徽章可高亮显示该作者的所有帖子。

URL 自动链接

在设置中启用此功能，可将帖子中的普通链接转换为可点击的锚点，因为 4chan 默认将其关闭。

自定义主题

统一的深色或浅色主题，可重新设计原生 4chan 论坛、目录和回复表单。

通用应用

一次购买即可在 Mac、iPad 和 iPhone 上使用，无需分级、订阅或应用内购买。

即时开关

主开关配合每个模块的独立开关，无需卸载即可关闭不需要的功能。

更新新功能

首次更新后的首次启动时，欢迎界面会显示更新内容，并在新增的设置项旁显示“NEW"标签。

工作原理

三步打造更好的 4chan

1

从 App Store 安装

下载 Clover Crow，一次购买即可通过家庭共享在 Mac、iPhone 和 iPad 上使用。

2

在 Safari 中启用

在 Safari 设置中打开“扩展”，启用 Clover Crow 并授予其对 4chan 的访问权限。

3

打开任意面板

像往常一样浏览 4chan，Clover Crow 接管预览、监视、快速回复等功能，并在设置中调整所有选项。

为什么选择 Clover Crow

Safari 原生完整工具包

专业用户优先

Clover Crow 专为 Safari 优化，集成了久经考验的浏览器高级工具包，包括引用预览、监视器、快速回复和过滤器。

零数据收集

无需账户、无需遥测、无需云端，您关注的线程、筛选条件和偏好设置均完全存储在本地设备上。

完全本地化

已翻译为 32 种非英语语言，可在应用内切换，且独立于系统语言。

需要 macOS 14.0（Sonoma）或更高版本，或 iOS/iPadOS 17.0 或更高版本，并配合 Safari 17+ 使用。
